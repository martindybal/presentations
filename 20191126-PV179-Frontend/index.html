Responsivní design prakticky

media query

css se aplikují pouze, pokud je spněna podmínka

SCSS: 

.about{
    width: 20vw;

    @include res-max-mobile {
        width: 100vw;
    }
}
 

$width-mobile: 480px;

@mixin res-max-mobile {
    @media only screen and (max-width: $width-mobile) {
        @content;
    }
}
CSS: 

.about{
    width: 20vw;
}
@media only screen and (max-width: 480px) {
    .about{
        width: 100vw;
    }
}
